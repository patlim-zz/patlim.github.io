Positioning

What are the differences between relative, absolute, and fixed positioning?

Relative

Element with relative position, is positioned relative to the normal position.
Essentially, relative to it's static position (default position property).
The positioning is controled with attributes top, right, left and bottom.
As you can see in the example, top means distance from the top of the default position.

The red box on the right has been moved relative to it's normal position using the following:

.redbox {
    position: relative;
    left: 90px; (Moves right 90px)
    top 70px; (Moves the down 70px)
}

Normal Position

Left: 90px;
Top: 70px

Absolute

Elements with absolute positioning can be placed anywhere on the page
Positioning is also controlled using attributes top, right, bottom and left.
Elements will be positioned relative to their parent element.
If the element does not have a parent element, it will be positioned relative to the page.
I've got a little secret about the last example, I used absolute position for the red box
Used in tandem with relative, absolute positioning is extremely powerful.

In the example I have placed the boxes in various positions. Below shows the how the green box is positioned in the middle:

.greenbox {
    position: Absolute;
    left: 50%; (Position halfway from the left)
    top 50%; (Position halfway from the top)
}

Left: 0px;
Top: 0px

Right: 0px;
Bottom: 0px

Left: 50%;
Top: 50%

Fixed

For fixed positioning, the element is positioned relative to the viewpoirt aka. browser window.
For example, the navigation bar up top is an typical use of fixed positioning.
The element will maintain it's position even when scrolling

That's enough of an example, but for some consitency.
Here's an example:

.example {
    position: fixed;
    bottom: 25%;
    right: 25%;
}
