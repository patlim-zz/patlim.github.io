An analogy to describe the differences between HTML and CSS

Let’s say you’ve got an electric guitar and you pop it into the amp.

You can play solos, chords and pluck away, this is the HTML component.

Now CSS you add a pedal board, with this you’re adding effects like an echo, distortion, reverb and a whole lot of feeling.

CSS adds flavour, enhancing the HTML.

Explain the difference between accessing data from arrays and objects.

Accessing data in an array is a lot simpler.

Data is a list-like object where you can select elements using their index (where they are in that list) e.g. myArray[3] will give you the 4th element in the list.

Objects are similar, but the elements are pairs.

These pairs are broken down into key and value.

For example myFriend[name] or myFriend[age].
